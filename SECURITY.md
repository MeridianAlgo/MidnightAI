# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

MeridianAlgo takes security seriously. If you discover a security vulnerability, please do NOT open a public issue.

Instead, please report security vulnerabilities to security@meridianalgo.com. We will acknowledge your report within 48 hours and provide a timeline for resolution.
