# Contributing to MeridianAlgo Cryptobot

First off, thank you for considering contributing to MeridianAlgo! It's people like you that make this project better.

## Code of Conduct
By participating in this project, you are expected to uphold our Code of Conduct.

## How Can I Contribute?

### Reporting Bugs
* Use the GitHub issue tracker.
* Describe the bug in detail and provide steps to reproduce.

### Suggesting Enhancements
* Open an issue with the "enhancement" label.
* Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## Development Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `pytest`
